========================================
  合社JSON - 轻量、专注的现代化 JSON 编辑与校验工作台
========================================

GitHub: https://github.com/scutken/json-tools

本项目 fork 自 fevrax/json-tools
  (https://github.com/fevrax/json-tools)
在原项目基础上做了大量功能、样式与交互调整。

合社JSON 是一款功能全面的 JSON 在线处理工具，
为开发者提供高效、直观的 JSON 数据编辑和校验体验。

---- 核心功能 ----

1. 多视图 JSON 编辑器
   文本视图：基于 Monaco Editor 的专业代码编辑体验，
              支持语法高亮、自动补全
   差异对比视图：方便对比两份 JSON 数据的差异
   表格视图：以表格形式直观展示 JSON 数据结构

2. 字符解码器
   时间戳解码器：自动识别时间戳并转换为可读日期格式
   Base64 解码器：检测并解码 Base64 编码字符串
   Unicode 解码器：自动解码 Unicode 转义序列为可读字符
   URL 解码器：识别并解码 URL 编码字符串
   支持全局或按编辑器实例单独启用 / 禁用解码器

3. JSON 修复
   使用 jsonrepair 自动修复常见 JSON 格式错误
   支持大整数精确解析（lossless-json）

4. JSON 查询
   支持 jsonquery 查询语法，快速定位和提取数据

5. 便捷功能
   多标签页：支持同时打开多个 JSON 文件处理
   深色 / 浅色主题切换
   标签页历史记录
   本地存储自动保存
   自定义字体大小
   键盘快捷键（新建 / 关闭标签页等）

---- 技术栈 ----
React 19 + TypeScript + Vite + TailwindCSS + Monaco Editor

---- 部署方式 ----
支持静态站点部署、uTools 插件运行
